# Description

A simple blog managment platform.

# Precondition

- add `.env.development` and `.env.production` in root directory

> which contains:

```
 VITE_API_URL = /api/
```

# Start

## Installation

```bash
$ pnpm install
```

## Running

```bash
$ pnpm run dev
```

## Build

```bash
$ pnpm run build
```

## Preview

```bash
$ pnpm run preview
```
